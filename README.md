# Jamie Booking One-Page Site

Polished one-page static site for dance studios to book Jamie for competitive choreography and workshops.

## Stack

- Plain HTML/CSS/JS
- Runs with Python's built-in static server through `npm start`; no npm registry install required.

## Files

- `index.html`: single-page structure and copy
- `styles.css`: premium black/white visual system and responsive layout
- `app.js`: mobile nav behavior + choreography estimate calculator
- `smoke-test.js`: required-content and no-CDN checks + JS parse check

## Run locally

```bash
npm start
```

Open `http://localhost:5210`.

## Test

```bash
npm test
```
